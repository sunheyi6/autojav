# AutoJava CLI 版本说明

## 版本 1.0.0

### 发布日期
2026-02-01

### 主要功能

1. **代码审计与自动修复**
   - 基于JavaParser解析Java源码，识别代码结构与潜在问题
   - 集成阿里巴巴Java开发规范的静态审计规则
   - 对接大模型API进行深度代码分析
   - 支持AI生成修复方案，预览和一键替换源码
   - 自动生成备份文件，确保代码安全

2. **自动接口文档生成**
   - 解析Spring系列注解，提取接口路径、请求方式、参数、返回体信息
   - 支持生成Markdown格式文档
   - 支持生成OpenAPI 3.0规范格式文档（YAML和JSON）
   - 支持文档导出和导入

3. **配置管理**
   - 支持本地和全局配置
   - 支持大模型API密钥配置
   - 支持配置的查看、设置和管理

4. **许可证管理**
   - 支持免费版、团队版、企业版、买断版四种版本
   - 基于版本的功能权限控制
   - 支持许可证激活和验证

### 技术特性

- **跨平台兼容**: 支持Windows、Linux、MacOS
- **Java版本兼容**: 支持Java 8+，推荐Java 17
- **AI集成**: 支持国内大模型（通义千问、文心一言、讯飞星火、Kimi等）
- **性能优化**: 高效的代码解析和处理
- **安全可靠**: 本地处理代码，保障代码安全

### 使用方法

1. **安装**
   - 下载 `autojav-cli-1.0.0.jar` 文件
   - 确保已安装Java 17或更高版本

2. **运行**
   ```bash
   java -jar autojav-cli-1.0.0.jar --help
   ```

3. **配置**
   - 首次使用前配置AI API密钥
   ```bash
   java -jar autojav-cli-1.0.0.jar config set ai.api.key <your-api-key>
   ```

4. **功能示例**
   - 代码审计: `java -jar autojav-cli-1.0.0.jar audit <file-or-directory>`
   - 文档生成: `java -jar autojav-cli-1.0.0.jar doc <file-or-directory>`
   - 配置管理: `java -jar autojav-cli-1.0.0.jar config list`

### 注意事项

- 使用AI功能需要配置有效的API密钥
- 代码修复功能会修改原始文件，请谨慎使用
- 审计结果仅供参考，最终修复方案请由开发者确认
- 本工具仅在本地处理代码，不会上传代码到云端
